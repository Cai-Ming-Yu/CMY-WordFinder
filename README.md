# 彩銘羽 WordFinder
テキスト中の特殊文字をチェックする簡単な方法。

It searches the provided paths, finds which text files contain non-ASCII characters, and outputs the paths to the corresponding files.

## Usage
wordfinder <file or dir 1> <file or dir 2> <...>
